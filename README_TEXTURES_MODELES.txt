BastPlan Paysage v17 - Bibliothèque de textures et dossiers 3D

Cette version contient une structure plus complète :
- textures/eau/piscine, bassin, fontaine
- textures/revetements/pelouse, gravier, dolomie, ecorce, pave, pierre, terrasse, dalles
- textures/vegetation/haies, arbres, arbustes, massifs
- textures/clotures, constructions, equipements
- icons/... pour les futures miniatures de bibliothèque
- models/... pour les futurs modèles 3D GLB/GLTF

Les textures fournies sont des SVG procéduraux légers. Elles servent de base fonctionnelle immédiatement, sans télécharger de gros fichiers.
Tu peux ensuite remplacer un SVG par une vraie texture PNG/WebP en gardant le même nom ou en modifiant textureAssetPaths dans app.js.

Pour la suite, les modèles 3D externes seront à placer de préférence en .glb dans models/<categorie>/.
