Aide rapide

Les bases pour utiliser BastPlan.

  • Sélectionnez un outil à gauche, puis dessinez sur le plan.
  • Utilisez “Sélection / déplacement” pour déplacer, modifier ou redimensionner un objet.
  • Le clic droit ouvre le menu : copier, dupliquer, verrouiller, supprimer.
  • Ctrl + glisser ajoute une zone, Alt + glisser retire une zone.
  • Double-clic : édite ou termine une forme libre / une courbe.
  • Utilisez le panneau de droite pour entrer des dimensions précises.
  • Le zoom se règle avec la barre ou Ctrl + molette.
